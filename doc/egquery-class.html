egquery class

Description

egquery is an S4 class that extends the eutil-class. This class provides a container for data retrived by calls to the NCBI EGQuery utility.

Details

eqquery objects have two slots in addition to the slots provided by basic eutil-class objects: termThe search term passed on to the EGQuery Utility countA data frame with the following columns: dbName, menuName, count, and status.

See also

egquery for generating calls to the NCBI ELink utility.
