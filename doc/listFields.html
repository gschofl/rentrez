Retrieve field codes for an NCBI database

Usage

listFields(db, all = FALSE)

Arguments

db
An NCBI database
all
if FALSE only field names, full names, and field descriptions are returned.

Description

Retrieve field codes for an NCBI database

Value

A data frame

Examples

pmf <- listFields("pubmed")
