Retrieve the number of records in an Entrez database matching a text query

Usage

ecount(term, db = "nuccore", ...)

Arguments

term
A valid NCBI search term.
db
An NCBI database (default = "pubmed").
...
Additional parameters passed on to esearch.

Description

Some additional details about this function

Value

A numeric.

Examples

## retrieve the number of records for a search term in a database
n <- ecount(term="Chlamydia psittaci", db="gene")


ESearch query using the ‘gene’ database.
Number of hits: 1092
