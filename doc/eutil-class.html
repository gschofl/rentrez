eutil class

Description

eutil is an S4 class that is extended by the einfo-class, esearch-class, esummary-class, efetch-class, and epost-class classes.

Details

These classes provide containers for the data returned from calls to the NCBI Entrez Utilities. Detailed information about the EUtilities provided by NCBI is available here.

Eutil class objects have three slots: urlA character vector containing the URL submitted to Entrez. errorAny error messages parsed from the output of the call submitted to Entrez. dataAn XMLInternalDocument-class or character vector holding the unparsed output from the call submitted to Entrez.
