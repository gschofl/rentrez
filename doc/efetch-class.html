efetch class

Description

efetch is an S4 class that extends the eutil-class. This class provides a container for data retrived by calls to the NCBI EFetch utility.

Details

efetch objects have three slots in addition to the slots provided by basic eutil-class objects: databaseThe name of the queried database. typeA character vector specifying the record view returned, such as Abstract or MEDLINE from PubMed, or GenPept or FASTA from protein modeA character vector specifying the data format of the records returned, such as plain text, HMTL or XML

See also

efetch for generating calls to the NCBI EFetch utility.
