esearch class

Description

esearch is an S4 class that extends the eutil-class. This class provides a container for data retrived by calls to the NCBI ESearch utility.

Details

esearch objects have eight slots in addition to the slots provided by basic eutil-class objects: databaseThe name of the queried database. countAn integer giving the total number of hits for a database query. retMaxAn integer giving the number of hits retrieved. retStartAn integer giving the index of the first hit retrieved. queryKeyParameter specifying a UID list attached to a user's Web Environment on the Entrez history server. webEnvParameter specifying the Web Environment on the Entrez history server containing the list of UIDs matching an Entrez query. queryTranslationA Character vector containing the search term as translated by the Entrez search system idListA Character vector containing the UIDs returned

See also

esearch for generating calls to the NCBI ESearch utility.
